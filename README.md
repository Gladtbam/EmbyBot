## Emby是什么？

Emby是一个主从式架构的媒体服务器软件，可以用来整理服务器上的视频和音频，并将音频和视频流式传输到客户端设备。

简单来说：Emby是一个私有的流媒体播放平台，类似爱奇艺奈飞等。
